Python implementation of velocity-based Change-point Inference (vMCI) algorithm :cite:p:`Senger2014`, :cite:p:`Gutzeit2022`. The algorithm automatically segments
human demonstrations of manipulation movements based on the velocity of the hand. Resulting segments correspond to movement 
building blocks charactericed with a bell-shaped velocity profiles.
